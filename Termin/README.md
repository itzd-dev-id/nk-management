# Struktur Folder Termin - NK Management

Folder ini berisi struktur dokumentasi termin proyek yang diorganisir berdasarkan kategori pekerjaan.

## Struktur Folder

```
Termin/
├── 01_Persiapan/
├── 02_Tanah/
├── 03_Struktur_Bawah/
├── 04_Struktur_Atas/
├── 05_Arsitektur/
├── 06_Atap/
├── 07_MEP/
├── 08_K3_dan_Keselamatan/
├── 09_Lansekap_dan_Lapangan/
├── 10_Material_On_Site/
├── 11_Dokumentasi/
└── 12_Furniture/
```

## Deskripsi Kategori

### 01. Persiapan
Pekerjaan persiapan lahan dan fasilitas proyek (Direksi Keet, Barak Pekerja, Mobilisasi, dll)

### 02. Tanah
Pekerjaan galian, urugan, dan pemadatan tanah

### 03. Struktur Bawah
Pekerjaan pondasi, pile cap, dan struktur bawah tanah

### 04. Struktur Atas
Pekerjaan rangka bangunan, kolom, balok, dan plat lantai

### 05. Arsitektur
Pekerjaan finishing dinding, lantai, plafon, dan kusen

### 06. Atap
Pekerjaan konstruksi dan penutup atap

### 07. MEP
Pekerjaan mekanikal, elektrikal, dan plumbing (Listrik/Air)

### 08. K3 dan Keselamatan
Pekerjaan keselamatan konstruksi (SMKK)

### 09. Lansekap dan Lapangan
Pekerjaan luar bangunan, lapangan olahraga, dan taman

### 10. Material On Site
Logistik dan kedatangan material di proyek

### 11. Dokumentasi
Pekerjaan pelaporan dan dokumentasi visual proyek

### 12. Furniture
Pekerjaan pengadaan mebel dan interior

---

## Cara Penggunaan

1. Setiap folder kategori dapat diisi dengan:
   - Foto progress pekerjaan
   - Dokumen termin
   - Laporan detail pekerjaan
   - Berita Acara

2. Gunakan nama file yang konsisten:
   - Format: `YYYY-MM-DD_Nama_Pekerjaan_Gedung_Kode_Progress_Seq.ext`
   - Contoh: `2026-02-11_Pengecoran_Kolom_Rusun_A_50%_001.jpg`

3. Untuk termin, kelompokkan file berdasarkan tanggal atau periode termin.
